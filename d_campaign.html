貝萊德基金作為一個強檔基金，強勢登場！

The project

鉅亨基金交易平台設立一個活動網頁，為貝萊德基金的強勢推出造勢，視覺與切角結合時事熱點：漫威電影 - 復仇者聯盟，以插畫的手法增加親切感與吸睛度 Web Demo Link。

My role

  • UI Design
  • 競品與風格分析
  • Icon 元件設計
  • Cowork with develop & BD

Challenge

插圖由插畫師製作，我的職責在於統整專案英雄聯盟發想與插畫師的畫風，進行 Campaign Site 的 UI 設計。
著重於介面視覺的挑戰，從對焦「漫畫風」到符合整體專業形象，進行針對性的風格嘗試與調整。

  • 定義「何謂漫畫風」，視覺的形容詞有時過於抽象，幾回視覺風格嘗試對齊方向。
  • 在「漫畫親切感」與「基金信任穩重感」兩者之間，如何在視覺傳達上做平衡？？

Design Strategy

初步視覺對焦

英雄聯盟，美漫的權威代表

漫畫風有多種表現手法，手繪的插畫風顯得活潑可愛，從何下手才能替基金形象傳遞以「信任、專業」為基石的調性？
嘗試兩種不同的「漫畫風」：A版同插畫感較親和力的漫畫風，B版強調電影氛圍的漫威視覺感。最後以B版作延伸發展。


活潑不失專業

延伸設計角色專屬 ICON

傳統的美式漫畫流於俗氣，滿版視覺逐步調整「具專業感」的深色調，背景式漸層壓圖等較大器的手法；漫畫巧思設計在背景圖型元素， 傾斜式按鈕與鮮豔的顏色帶出活潑感。


統整延續品牌形象

以 Brand 青綠色點綴細節、貫穿全站

和總網站「鉅亨基金交易平台」的 Logo 與 Branding 色對齊，在細節上凸顯總網站的 Brand 青綠色，整合色調風格，維持一體性且延伸形塑總官網的品牌形象。


Learning

Web UI Design Thinking

風格與形象的塑造，會因取角方向而有大大的不同，Campaign Site 就像一個正在說故事的線上展演場，除了整體視覺感外，更多的 UI 思考是：如何在「吸睛張力、品牌形象、閱讀舒服感」三者之間調配與權衡，細細雕琢展演館的佈置，讓故事越真實越有看頭。
