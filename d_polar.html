沈浸式體驗互動，一窺北極拉雪橇的極地生活！

The project

2017選上作為台灣代表，前往參與北極拉雪橇活動。結束後決定將特殊體驗以同樣「沈浸式且深刻」的方式回饋支持者，視覺化的手法整合「影、音、圖、文」，結合影片網頁技術，以互動式網頁呈現最終成果 Polar Web Link。

Challenge

「去北極拉雪橇」本身是一個極為特殊的體驗活動，希望能打造一個不需要有講者，使用者也能透過該平台，深入了解體驗並感同身受。

  • 如何將這份全方位衝擊的特殊感受，盡可能，「真實的」傳達給使用者？
  • 如何將爆炸般的巨額新知量，以有效率又好吸收，且不中斷體驗的方式呈現？

Design Strategy

如何創造「沈浸式」體驗？

UX Usability

介面的按鈕體驗與反饋設計，何時會想繼續播放，何時會想進一步對極地或雪橇有更多的了解？
以情境影片創造投入感，類遊戲的互動介面，加強互動體驗，以第一人稱視角，貫穿「沈浸式」的視角與介面。


User Journey

重新解構故事架構，將五天的行程與準備，濃縮成一日極地生存法則，帶領使用者進入故事。腳本設計細節：將旅程來到和結束的心境，與一天的開始和日暮相呼應。


UI Design

以不中斷體驗作介面設計的核心，將龐雜的影音圖文資訊，以時間節點的方式呈現，並重新設計圖文的閱讀版面。
